# Microposts

A front-end CRUD application that allows performing CRUD operation through json-server simulating an API.

### Version
1.0.0

## Usage

### Installation
Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```

### API
Launch json-server

```sh
$ npm run json:server
```
